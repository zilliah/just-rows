Saved Counters

This simple row counter will count and remember and THAT'S IT.

Your count will be saved as long as you're using the same device and browser. You can edit the counter to add pattern repeats and a counter name, or to start at a particular row number.

You can have as many row counters as you like, but I haven't tested more than 5. If things get weird, you can contact me.

I am so annoyed at row counters that try to be project trackers and make things way more complicated than they need to be. I just want a simple row counter that doesn't require any set up or installing apps or similar nonsense, so here it is.
